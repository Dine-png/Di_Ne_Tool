# VRM feature attribution

The simplified VRM preparation workflow in Di Ne Tool was developed with permission from
Kuroiine Ushina and is inspired by **VRM Supporter** (Copyright © 2026 Kuroiine Ushina).

This integration is intentionally not a copy of the original full pipeline. It contains only
automatic outfit-bone merging, VRC PhysBone to UniVRM 0.x SpringBone conversion, constraint
substitution, and VRM-oriented component cleanup.
